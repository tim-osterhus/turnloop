# Project Outline

## Corebound:
- Goal: a browser-based mining game inspired by Motherload.
- Status update: the foundation implementation (Vite/React/TypeScript scaffold, fixed-timestep movement simulation, and drill/collision physics), deterministic seeded world generation and ore-placement logic, economy/base interaction with upgrade progression, save-state persistence with onboarding tutorial prompts, HUD feedback, and initial hazard systems were completed on 2026-03-07.
- Public repo link: `https://github.com/tim-osterhus/corebound`
- Local repo location: `corebound/`
  - Sits inside the base `turnloop/` repo as its own project directory, and should be treated as such.
- Additional information should be added here.
