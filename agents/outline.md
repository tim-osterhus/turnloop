# Project Outline

## Corebound:
- Goal: a browser-based mining game inspired by Motherload.
- Status update: the foundation implementation (Vite/React/TypeScript scaffold, fixed-timestep movement simulation, and drill/collision physics) was completed on 2026-03-07; deterministic seeded world generation and ore-placement logic were also completed in the same day.
- Public repo link: `https://github.com/tim-osterhus/corebound`
- Local repo location: `corebound/`
  - Sits inside the base `turnloop/` repo as its own project directory, and should be treated as such.
- Additional information should be added here.
